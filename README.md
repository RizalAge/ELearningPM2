# ELearningPM2

Aplikasi Flutter untuk mata kuliah **Pemrograman Mobile 2**.  
Project ini merupakan latihan penggunaan **Flutter + SQLite**.

---

## Fitur Aplikasi
- CRUD Notes (Tambah, Baca, Edit, Hapus)
- Penyimpanan lokal (SQLite)
